Queues

A queue is another linear data structure in C++ that follows the First-In-First-Out (FIFO) principle. It can be thought of as a collection of elements with two primary operations: enqueue and dequeue.

Elements are added to the back of the queue using the enqueue operation, and they are removed from the front of the queue using the dequeue operation.

Common Uses of Queues in C++

  • Task Scheduling: Queues are often used in task scheduling algorithms. In real-time systems, processes or tasks are added to a queue based on their priority or arrival time. The task with the highest priority or the first to arrive is dequeued and executed next.
  • Breadth-First Search (BFS): In graph algorithms, BFS uses a queue to explore and traverse all nodes at the current level before moving to the next level. This is useful for finding shortest paths in unweighted graphs and solving various graph-related problems.
  • Print Queue: In printer management systems, print jobs are often stored in a queue. The jobs are processed in the order they are received, ensuring fairness in printing and preventing resource contention.
  • Task Management in Multithreading: Queues are valuable for managing tasks in a multithreaded environment. Worker threads can dequeue tasks from a queue and execute them concurrently, ensuring proper synchronization and load balancing.
  • Buffering and Data Streaming: Queues can be used to buffer data in streaming applications. Data producers enqueue data, and consumers dequeue and process it. This ensures a smooth and controlled flow of data.

Using Queues in C++

In C++, queues can be implemented using various data structures, such as arrays, linked lists, or the standard library's std::queue container. Here's a basic example using std::queue: 


        #include <iostream>
        #include <queue>

        int main() {
            std::queue<int> myQueue;
    
            // Enqueue elements
            myQueue.push(10);
            myQueue.push(20);
            myQueue.push(30);
    
            // Dequeue elements
            while (!myQueue.empty()) {
                std::cout << myQueue.front() << " "; // Print front element
                myQueue.pop(); // Remove front element
            }
    
            return 0;
        }
