Module 9

Binary Trees and Algorithm Design Patterns

 

Algorithm Design and Complexity

In your study of various programming languages, you have seen that building programming solutions often comes down to applying familiar design patterns to common problems. Handling data structures is no different.

In this module, consider some common patterns and take a closer look at a few of those and then apply that to building a Binary Search Tree (BST).

Remember that algorithm complexity, and efficiency, is expressed as a function of the size of the data set.

 

You'll have two weeks for this module, be sure to start the discussion assignment in the first week.

In this module, you will:

  • Read, investigate, and discuss algorithm design topics
  • Implement a binary search tree
