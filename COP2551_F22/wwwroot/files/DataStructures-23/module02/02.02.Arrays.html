C++ Arrays and Their Properties

An array is a collection of elements of the same data type, stored in contiguous memory locations. Arrays provide a convenient way to store and manipulate multiple values under a single identifier.

Declaring and Initializing Arrays in C++

To declare an array, you specify the data type of its elements, followed by the array's name and its size in square brackets. 


// Declaration and initialization of an integer array
int numbers[5] = {10, 20, 30, 40, 50};

You can also omit the size when initializing, and the compiler will automatically determine it based on the number of elements: 



// Declaration and initialization without specifying size
char vowels[] = {'a', 'e', 'i', 'o', 'u'};

Accessing Array Elements

Array elements are accessed using zero-based indexing. The first element is at index 0, the second at index 1, and so on.


int thirdNumber = numbers[2]; // Accessing the third element (30)
char firstVowel = vowels[0];  // Accessing the first element ('a')

Array Properties

Arrays have several important properties:

  • Size: The number of elements in the array.
  • Contiguity: Array elements are stored in adjacent memory locations.
  • Fixed Size: Once an array is declared, its size is fixed and cannot be changed.

Iterating Through Arrays

Loop structures like for or while can be used to iterate through array elements:


for (int i = 0; i < 5; ++i) {
    // Access and manipulate numbers[i]
}

int index = 0;
while (index < 5) {
    // Access and manipulate vowels[index]
    ++index;
}

Multi-dimensional Arrays

C++ also supports multi-dimensional arrays, allowing you to create matrices and tables:


int matrix[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};

Arrays are a fundamental data structure in C++, providing a versatile way to store and manage collections of values.
