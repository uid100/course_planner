C++ Vectors: Dynamic Arrays

In C++, a vector is a dynamic array that provides more flexibility and functionality compared to traditional arrays. Vectors are part of the Standard Template Library (STL) and offer a convenient way to work with collections of elements while automatically managing memory allocation and resizing.

Introduction to Vectors

C++ vectors are containers that can dynamically adjust their size to accommodate the number of elements they hold. Unlike fixed-size arrays, vectors can grow or shrink as needed, making them highly versatile for various programming tasks.

Creating and Initializing Vectors

Creating a vector involves including the <vector> header and specifying the data type of the elements it will hold:


#include <vector>
using namespace std;

// Creating a vector of integers
vector<int> numbers;

Vectors can also be initialized with values using an initializer list:


// Initializing a vector with values
vector<double> prices = {10.99, 20.49, 5.99, 8.75};

Adding and Removing Elements

Vectors provide methods for adding and removing elements:


// Adding elements to the end of the vector
numbers.push_back(42);
numbers.push_back(99);

// Removing the last element
numbers.pop_back();

// Inserting an element at a specific position
numbers.insert(numbers.begin() + 2, 73);

// Erasing an element at a specific position
numbers.erase(numbers.begin() + 1);

Accessing Vector Elements

Vector elements can be accessed using zero-based indexing, just like arrays:


int secondNumber = numbers[1]; // Accessing the second element
double lastPrice = prices.back(); // Accessing the last element

Vector Properties

Key properties of vectors include:

  • Dynamic Resizing: Vectors automatically resize when elements are added or removed.
  • Efficient Memory Management: Memory allocation and deallocation are handled by vectors.
  • Range Checking: Vectors perform bounds checking to prevent invalid access.

Iterating Through Vectors

Loop structures can be used to iterate through vector elements:


for (size_t i = 0; i < numbers.size(); ++i) {
    // Access and manipulate numbers[i]
}

for (double price : prices) {
    // Access and manipulate each price
}

C++ vectors offer a convenient and efficient way to work with dynamic collections of elements, making them a valuable tool in modern programming.
