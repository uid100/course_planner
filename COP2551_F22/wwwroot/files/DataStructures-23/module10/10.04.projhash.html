Project: Hash tables

Purpose

Understand the use of hash tables, by constructing hash tables from a simple data set.

 

Instructions

Using the values provided below, show the hash key and what the hash table will look like when using the two different techniques indicated.

The hash function used is based upon the first letter of the person's last name.

  • The array size is 27
  • Index 0 is not used
  • The letter 'A' will go to index 1, 'B' to index 2, etc

  1. Separate chaining
  2. Open addressing with linear probing
     
      i_table = ( h(k) + j ) mod S
     
    where i is the index of the underlying array,
    h is the hash function,
    j is the iteration of the probe, and
    S is the size of the table
