File Extensions

I don't know why this is not the default setting anymore, but on both Windows and MacOS, it is not. As a developer, you need to see this information.

MacOS:

Select Finder > Preferences from the menu and check the box.

Windows 11:

From File Explorer: View > Show > File name extensions.
(Windows 10: it's a checkbox on the toolbar)

Select Finder > Preferences from the menu and check the box.
