Cloud Deployment Types

 

Four cloud deployment models are widely recognized: public clouds, private clouds, community clouds, and hybrid clouds.

A cloud deployment model is defined according to where the infrastructure for the deployment resides and who has control over that infrastructure.

This brief article describes a fifth model https://www.geeksforgeeks.org/cloud-deployment-models/.

Questions to consider:

  • What is the relationship of overhead to deployment model?
  • Which do you think represents the most secure? Is it a different answer for different applications?
  • Aside from initial investment, what are the factors which might drive continuing operating costs for each one?
  • What are compelling reasons for selecting one over another?
  • Can you think of applications which would be best-fit for each of the four (or five)?
