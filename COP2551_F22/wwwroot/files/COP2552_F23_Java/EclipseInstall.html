This is a new install. For this installation, I began by creating a virtual machine running Windows 11. It has no other software installed on it.

  1. First, search for "eclipse download." Microsoft Edge is the default browser installed with Windows.
  2. From the Eclipse Downloads page, select the download link for your processor.
  3. Launch the downloaded Installer
  4. Select the installation for Java Developers
  5. The installation did not take long, even on this minimalist VM.
  6. Once the installer completes, launch the application and select the working directory.
  7. When the application launches, a new project can be created from the Welcome screen, or close it and use the menu to create a new Java project.
  8. From the new project, right-click the src folder and add a new class.
  9. Add the code and run. In the example, the project name and the class name do not match. To keep things simple, delete the module-info.java file and the code will run as is.
