Most of you know this, since it was discussed on the Discord channel too. But in case you missed it:

Integrated Development Environments (IDEs) are an important tool to the developer.

The IDE combines a set of tools into a single environment. Usually an editor, a compiler, a runtime debugger, a profiler and others.

Combined (integrated) into a single toolbox (environment) for creating (development) coded applications.

Some of these are really cool. All of these are designed and intended to make the activity of writing software applications easier and more streamlined. Eclipse for Java is no different. It is a good tool and like any tool, the more you use it, the more familiar and the more useful it will be.

 

I have been told that there is a new release for Eclipse which is causing some students challenges just getting it to install.

That's ok.  (Wait, what?)

Yeah. For a couple of reasons.

 

First, I'm ok with letting students struggle with something on their own to figure something out. But only for a little while. It is not my intent or desire to drive anyone to the point of complete frustration.

I will admit that I haven't installed the newest Eclipse in several months. But I will. It might not be for a few more days, but I'll try and get updated installation instructions to you by next week.

 

And Second, the other reason I'm not too worried about the Eclipse install just yet is that, at least for the first project, you won't really need it. Java source code is written in a text file. That's all. I encourage you to write your source code for this first project using your favorite text editor and calling the compiler from the command line. 

javac MyBrainSurgeryCheatApp.java

 

and then calling the compiled class with the java runtime 

java MyBrainSurgeryCheatApp

 

Since the app that you are writing is probably contained in a single class, this will more clearly demonstrate that the IDE is not a magic wand to create applications. It will help demonstrate just what is required to make your application (class) work. And it will give you something to compare with when you do have the Eclipse IDE installed.

 

For those of you who had no trouble installing Eclipse, great. Full speed ahead!
