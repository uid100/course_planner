Specifications

Write a Java program using good object-oriented programming principles that will demonstrate the use of multiple classes.

For any person with an active lifestyle and trying to maintain weight, keeping up with the number of minutes exercising and the number of calories burned is important.

This project will keep track of a person's activity in minutes and calculating the number of calories burned while exercising.

The number of calories burned depends on several factors such as the weight of the individual, duration, and intensity. To keep this project within scope, only a few options will be available in which to choose.

When the application begins, the number of minutes exercised and the number of calories burned will be 0. The user will continue to be prompted what they have completed until the user wishes to exit.

Give the user a menu of various types of exercises as shown in the example below. From there, prompt the user for the intensity level used when exercising.

Number of calories consumed per minute:
Activity Biking Running Swimming Walking Weight Training
Moderate 10 12 6.8 5 3.7
Vigorous 14.3 17 14.8 5.6 5.9
If the user presses 1:
The Bicycle class will be utilized.
    It should:
  • prompt the user for the amount of time riding
  • calculate the number of calories consumed and add to the total
  • return back to the main menu
If the user presses 2:
The Run class will be utilized.
    It should:
  • prompt the user for the amount of time running
  • calculate the total number of calories consumed
  • return back to the main menu
If the user presses 3:
The Swim class will be utilized.
    It should:
  • prompt the user for the amount of time swimming
  • calculate the total number of calories consumed
  • return back to the main menu
If the user presses 4:
The Walk class will be utilized.
    It should:
  • prompt the user for the amount of time walking
  • calculate the total number of calories consumed
  • return back to the main menu
If the user presses 5:
The Weights class will be utilized.
    It should:
  • prompt the user for the amount of time lifting
  • calculate the total number of calories consumed
  • return back to the main menu
If the user presses 6:
The ExitProgram class will be utilized.
    Display in a message dialog box the aggregates of:
  • the total amount of time spent and the total number of calories consumed cycling
  • the total amount of time spent and the total number of calories consumed running
  • the total amount of time spent and the total number of calories consumed swimming
  • the total amount of time spent and the total number of calories consumed walking
  • the total amount of time spent and the total number of calories consumed weight-lifting
  • End the program
