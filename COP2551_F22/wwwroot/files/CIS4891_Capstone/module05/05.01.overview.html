Data-Driven Design

Data plays a crucial role in software design as it helps developers make informed decisions about the design and functionality of the software they are creating.

Starting with the data design helps to identifying user needs and preferences. By analyzing user data, developers can gain insights into what users want and need from the software. This can inform decisions about which features to include and how to design the user interface.

Recall from the discussion of REST APIs, the data response is a representation of a data entity. There may be multiple visualizations for any given set of data entities depending on purpose and perspetive. Having well-designed data relationships will help to make the data more usable and useful.
