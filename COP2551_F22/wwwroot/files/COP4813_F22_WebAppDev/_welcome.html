Web Application Programming

 

COP2552 - Fall 2022

This course is designed to teach students advanced concepts in developing ASP.NET Core MVC, database driven web apps using HTML/CSS and C# in the Visual Studio IDE.

    Topics covered include:
  • Endpoint routing, Razor views
  • Model binding
  • Data validation
  • Dependency injection
  • Bootstrap for responsive design
  • EF (Entity Framework) Core for database handling

 

Student Learning Objectives

  • ✔ Develop a single-page MVC web app
  • ✔ Develop a data driven MVC web app
  • ✔ Use Bootstrap to develop a responsive MVC web app
  • ✔ Test and debug a MVC web app
  • ✔ Work with controllers and routing, session state and cookies, model binding, and razor views
  • ✔ Use EF Core to work with data in a database
