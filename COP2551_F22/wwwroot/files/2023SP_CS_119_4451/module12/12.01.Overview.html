Chapter 12
Manipulating Larger Quantities of Data

Data collections are used in programming to store and manipulate multiple values of the same or different data types. Data collections are a fundamental concept in programming, as they enable programmers to organize data in a way that can be easily accessed and manipulated.

Data collections can be used for

  • Storing lists of data
  • Grouping related data
  • Managing large datasets
  • Implementing data structures
