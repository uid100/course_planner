Using Numbers

Open IDLE and create a new file. Save it with a meaningful name, such as helloworld.py. The .py extension is important.

Copy the following code into your file, save it and run it. 

#!/usr/bin/env python 3

#Chapter 2 add 2 numbers demo

# from PLD page 33, an identifier is a class, method or variable name
# In Python, the typical naming convention is lower case with underscore _
# between words to improve readability
# The data type is determined by the initial value assigned
# Note we use a single = for assigning values in Python

# Declare variables
value1 = 0
value2 = 0
total = 0

# get input
value1 = int(input("Enter first value: "))
value2 = int(input("Enter second value: "))

# process
total = value1 + value2

# output - display total
print("Total = " + str(total))
