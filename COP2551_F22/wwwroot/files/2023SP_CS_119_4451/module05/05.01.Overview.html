Chapter 5
Arrays

    Content:
  • Storing data in arrays
  • Parallel arrays
  • Looping through arrays

 

With chapter 5, we move into a data structure called arrays. Arrays are very simply a contiguous series of variables in memory with the same name and data type. We access each element or variable by using a subscript or index. Head's up, Python is a little bit different here. In Python, arrays are not built-in or native so we need to import an array library. The native and more preferred data structure in Python is the list. It works just like an array but allows different data types. While not 100% correct, we are going to treat Python lists basically like arrays.

    At the end of this module, you will be able to:
  • Demonstrate the use of arrays
  • Demonstrate indexing values from parallel arrays
