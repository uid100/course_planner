Using Python and IDLE

In just about every programming language, we start out with a simple Hello World program.

This gives us a chance to become familiar with the IDE, some simple code and running a program.

Start out by downloading and installing Python.

Python Download

After you install Python to your system, run the following command to make sure everything is working correctly. 

python3 --version

Hello World

Open IDLE and create a new file. Save it with a meaningful name, such as helloworld.py. The .py extension is important.

Copy the following code into your file, save it and run it. 

#!/usr/bin/env python3

# Lines starting with a hash (#) are comments so this is a comment
print("Hello world!")

What happened?

The first line also contains a comment, but this is a special line which needs to be on all of your Python code. It is called a compiler directive and tells the operating system to run the program using Python.

This line is called the shebang! and we'll talk more about it later.
